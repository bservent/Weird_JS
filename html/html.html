HTML Overview

What is HTML?

Hypertext Markup Language can be considered as the backbone of the Web. Web pages get there structure from it where CSS and Javascript are more to do with the styling and presentation. Hypertext refers to links that connect the web.HTML is made up of elements that are defined by different tags. Examples of some are head, title, body, header, footer, article, section, p, div, span, img, aside, nav, ul, ol, li, etc.

Dissecting HTML boilerplate

!DOCTYPE html opening tag is required to denote that the file is html and is running correctly. The root element is html and it wraps all the other elements in the file. The head contains all the stuff you want to include on the page that isnt content. This includes links and scripts to js and css, along with keywords for search results. Now on to the meta data thats displayed in the head. Meta charset="utf-8" sets the character used in document. Title sets the page title and will display in browser tag. Then onto the body where the content will be displayed
